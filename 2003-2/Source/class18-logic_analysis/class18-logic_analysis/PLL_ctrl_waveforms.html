Sample behavioral waveforms for design file PLL_ctrl.v

The following waveforms show the behavior of altpll megafunction for the chosen set of parameters in design PLL_ctrl.v. The design PLL_ctrl.v has Cyclone AUTO pll configured in NO_COMPENSATION mode The primary clock input to the PLL is INCLK0, with clock period 40000 ps. Output port LOCKED will go high when the PLL locks to the input clock.

Fig. 1 : Wave showing NO_COMPENSATION mode operation.

When input port ARESET is asserted, it will cause the LOCKED port and all CLK outputs to drop to zero. The PLL will relock to the input clock when this port is deasserted.
